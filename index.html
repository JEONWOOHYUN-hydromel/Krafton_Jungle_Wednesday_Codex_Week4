실제 영역

Actual DOM

선택한 테스트 케이스를 적용하면 Actual DOM이 갱신됩니다.

가상 DOM / Diff / Patch

이 영역은 실제 DOM 대상입니다. 오른쪽 테스트 영역에서는 먼저 미리보기 Virtual DOM을 만든 뒤, Patch로 변경된 부분만 적용합니다.

  • 브라우저 DOM을 Virtual DOM 트리로 읽기
  • 이전 / 다음 Virtual DOM 비교
  • 변경된 실제 DOM만 패치 적용
안정적인 key는 리스트 항목이 추가, 삭제, 재정렬되어도 identity를 유지합니다.
