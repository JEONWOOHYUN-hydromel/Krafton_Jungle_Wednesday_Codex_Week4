Guide

시연 포인트

  1. 문장 하나만 바꿔서 TEXT patch를 보여줍니다.
  2. data-version 같은 속성을 바꿔서 PROPS patch를 보여줍니다.
  3. feature-list 내부 li[data-key] 순서를 바꿔 REORDER patch를 보여줍니다.
  4. 새 li data-key="..." 를 추가하거나 제거해 keyed 구성 변경과 REORDER를 시연합니다.
  5. 루트 섹션에 새 <p> 를 추가하거나 삭제해 일반 CREATE / REMOVE도 확인합니다.
  6. 마지막에 Undo / Redo로 state history가 함께 바뀌는지 확인합니다.
