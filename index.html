Guide

Demo Flow

  1. Change one text node directly in the preview to show a simple TEXT patch.
  2. Delete one keyed list item and check whether the remaining keys stay preserved.
  3. Add a new line or paragraph in the preview to show CREATE / REMOVE behavior.
  4. Reorder keyed list content and compare Auto vs Index Only mode.
  5. Watch the inline badges, patch log, and VDOM tree update together.
  6. Use Undo and Redo to prove that history restores both panels.
